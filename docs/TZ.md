## 1. KONSEPT VA G'OYA

### 1.1 Asosiy G'oya

Oddiy AI chatbot emas — **Ixtisoslashtirilgan AI Bo'limlari Do'koni.**

Har bir foydalanuvchi o'z sohasini tanlab, **aynan o'sha sohaning eksperti bo'lgan AI** bilan ishlaydi. Bu AI o'sha soha bo'yicha:
- Barcha professional bilimlarga ega
- Sohaga oid maxsus asboblar (tools) ishlatadi
- Sohaga mos savollar beradi (AI Wizard)
- Sohaga xos hujjatlar, shakllar, hisobotlar yaratadi

### 1.2 Analogy

```
Eski dunyo:              Yangi dunyo (Sizning platforma):
─────────────────        ──────────────────────────────
Umumiy shifokor    →     Kardiolog + Nevrolog + Jarroh
Umumiy advokat     →     Soliq advokati + Korporativ + Jinoyat
Umumiy konsultant  →     Biznes + Buxgalter + Marketing eksperti
```

### 1.3 Platforma nomi (taklif)
**"SOHA.AI"** — Har bir sohaning AI eksperti

## 8. DATABASE SXEMASI

### 8.1 Yangilangan asosiy jadvallar

```sql
-- Foydalanuvchilar
users:
  id, email, password_hash, name
  role: ENUM('admin', 'teacher', 'user')
  active_department: VARCHAR  -- joriy bo'lim
  tariff: ENUM('free','basic','pro','max')
  tariff_expires: TIMESTAMP
  own_api_key: VARCHAR (encrypted)
  tokens_used_today: BIGINT

-- Bo'limlar konfiguratsiyasi
departments:
  id: VARCHAR PRIMARY KEY  -- 'accounting', 'coding'...
  name: VARCHAR
  system_prompt: TEXT
  tools: JSONB
  wizard_config: JSONB
  is_active: BOOLEAN
  sort_order: INT

-- Suhbatlar (bo'lim bilan)
conversations:
  id, user_id
  department_id: VARCHAR REFERENCES departments(id)
  title: VARCHAR
  model: VARCHAR
  type: ENUM('chat','image','video','tool')
  created_at: TIMESTAMP

-- Xabarlar
messages:
  id, conversation_id, role
  content: TEXT
  thinking: TEXT
  tokens_input, tokens_output: INT
  metadata: JSONB  -- rasm URL, tool result...
  created_at: TIMESTAMP

-- Shablonlar
templates:
  id, department_id
  name: VARCHAR
  category: VARCHAR
  content: TEXT  -- shablon matni
  variables: JSONB  -- to'ldirish kerak maydonlar
  usage_count: INT

-- Foydalanuvchi sozlamalari (bo'lim bo'yicha)
user_department_settings:
  user_id, department_id
  preferred_model: VARCHAR
  custom_instructions: TEXT  -- qo'shimcha shaxsiy ko'rsatmalar
  created_at: TIMESTAMP
```
